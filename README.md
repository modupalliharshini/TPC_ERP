# TPC-ERP Admin Dashboard

A high-fidelity, functional ERP (Enterprise Resource Planning) dashboard prototype built with **Next.js 14**, **TypeScript**, and **CSS Modules**. This platform features a premium design system tailored for modern school and organizational management.

## 🚀 Key Features

- **Personalized Experience**: Dynamic greetings and profile initials derived from login identity.
- **Student Management**: Registry for managing student records with real-time add functionality.
- **Staff Directory**: Functional faculty hiring and management with staff-strength tracking.
- **Fees & Finance**: Accounting module for recording fees with live "Total Collected" and "Outstanding Dues" metrics.
- **Attendance Tracker**: Class-wise attendance reporting with live percentage and status indicators.
- **Exam Section**: Complete scheduling workflow for upcoming and recent examinations.
- **Course Catalog**: Dynamic catalog management with category-specific iconography.
- **Library & Transport**: Tools for managing book inventories and bus route monitoring.

## 🛠️ Technology Stack

- **Framework**: Next.js 14.2.3 (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS Modules (Global ERP Form Design System)
- **Icons**: Lucide React
- **Optimization**: Next.js Image Optimization (`next/image`)

## 🎨 Global Design System
The project follows a unified design language:
- **Brand Colors**: Deep Navy (`#1e293b`), Vibrant Red (`#9f1220`), Slate Grey (`#64748b`).
- **Typography**: Outfit (Google Fonts).
- **ERP Form System**: A centralized set of utility classes (`.erp-form`, `.erp-input`, `.erp-btn-submit`) for pixel-perfect across all modules.

## 🏁 Getting Started

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/modupalliharshini/TPC-ERP.git
    cd TPC-ERP
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```

4.  **Production Build**:
    ```bash
    npm run build
    ```
